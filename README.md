# 💰 Personal Finance Analytics & Financial Behavior Research

This project presents a comprehensive data analysis and predictive modeling study on the financial behavior of university students. The research explores budgeting habits, spending patterns, savings behavior, and financial confidence using statistical analysis, machine learning, and NLP techniques.

---

## 🚀 Project Overview

The objective of this research is to analyze how demographic and behavioral factors influence personal financial management among students. The project combines:

* Financial behavior analysis
* Statistical modeling
* Predictive analytics
* NLP-based text analysis
* Data visualization techniques

The study aims to identify key factors affecting:

* Budgeting effectiveness
* Spending control
* Savings habits
* Financial confidence

---

## 📊 Dataset & Research Scope

The dataset contains responses from **121 university students** and includes variables such as:

* Age Group
* Gender
* Living Status
* Pocket Money Range
* Transportation Expenses
* Mobile/WiFi Bills
* Savings Behavior
* Budgeting Practices
* Spending Tracking
* Financial Confidence

---

## 🔍 Data Analysis & Methodology

### 🔹 Data Collection & Cleaning

* Survey-based dataset collection
* Data preprocessing & normalization
* Handling missing and inconsistent values

### 🔹 Exploratory Data Analysis (EDA)

* Descriptive statistics (Mean, Median, Mode)
* Correlation analysis
* Boxplot analysis
* Heatmaps & distribution graphs
* Financial behavior trend analysis

### 🔹 Predictive Analytics

A Linear Regression model was developed to predict financial confidence and spending behavior using multiple independent variables.

### 🔹 Model Evaluation Metrics

* Mean Absolute Error (MAE): **0.464**
* Mean Squared Error (MSE): **0.347**
* R-squared Score (R²): **0.71**

The model demonstrated strong predictive capability in understanding student financial behavior.

---

## 🧠 NLP & Text Analytics

The project also includes Natural Language Processing (NLP) techniques for analyzing survey suggestions and feedback.

### NLP Techniques Used

* Tokenization
* Stopword Removal
* Text Cleaning
* Bigram Network Graphs
* Word Cloud Generation

These techniques helped identify common financial concerns and behavioral patterns among students.

---

## 📈 Key Insights & Findings

* Students with higher pocket money generally spend more.
* Budgeting significantly improves spending control and financial confidence.
* Students who track expenses are more financially disciplined.
* Saving habits reduce the chances of running out of money before month-end.
* Living independently increases financial pressure and spending variability.
